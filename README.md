# Teste

Repositorio simples criado para testar a integracao do Codex com GitHub.

## O que tem aqui

- `index.html`: estrutura da pagina
- `styles.css`: estilos visuais
- `script.js`: interacao basica

## Como usar

Abra `index.html` no navegador ou publique com GitHub Pages.
